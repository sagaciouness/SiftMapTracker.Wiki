# 屏幕识别：五分钟快速上手

本页只覆盖第一次可用所需步骤。查蛋和童话事件共用 OCR 服务，但校准区域不同；先选择你要使用的功能。

## 1. 固定游戏画面

在开始前保持以下条件稳定：

- 游戏窗口位于准备长期使用的显示器；
- 分辨率、Windows 缩放和窗口位置不再调整；
- 目标界面没有被浏览器、聊天窗口或鼠标提示遮挡。

屏幕识别保存物理屏幕坐标。完成校准后再移动游戏窗口，会截取到错误位置。

## 2. 打开屏幕识别工作区

在客户端顶部工作区切换器中点击“屏幕识别”。左侧会显示独立导航：

- 查蛋控制台；
- 童话事件；
- 识别记录；
- 精灵数据库；
- 采集校准；
- 按键绑定。

## 3. 检查快捷键

进入“按键绑定”，确认：

| 功能 | 默认快捷键 |
| --- | --- |
| 查蛋识别 | `Alt + E` |
| 开始/暂停童话事件监测 | `Alt + T` |

状态必须显示注册成功。发生冲突时，改成未被其他程序占用的组合键并保存。

## 4. 选择一条上手路径

### 路径 A：完成第一次查蛋

1. 打开“采集校准”，选择“查蛋识别”。
2. 点击“依次框选两个区域”。
3. 第一步框住完整蛋名称和下方横线，并在右侧为长名称留出空间。
4. 第二步一次框住身高、体重及 `m`、`kg` 单位。
5. 回到游戏，打开一个蛋的详情页。
6. 按 `Alt + E`。

识别成功时：

- 灵动岛会从“查蛋待命”展开；
- 查蛋控制台显示蛋名称、身高、体重和匹配状态；
- “识别记录”新增一条北京时间记录。

结果为“多个候选”或“数值越界”不代表 OCR 一定失败。先核对控制台显示的原文，再决定是否重新框选。

### 路径 B：开始童话事件监测

1. 打开“童话事件”。
2. 搜索并选择目标精灵。
3. 点击“打开校准中心”，切换到“童话事件”。
4. 框住右上角完整精灵名称区域。
5. 框住完整结果横幅，确保血脉句子和“发现异色精灵！”都不会落在选区外。
6. 返回童话事件页，点击“开始监测”或按 `Alt + T`。

监测成功时：

- 页面状态变为“正在监测”；
- 灵动岛显示目标、当前次数和保底剩余；
- 地图追踪会停止；
- 再按 `Alt + T` 可暂停，暂停后不会自动恢复地图追踪。

::: warning
童话监测期间不能同时查蛋。按 `Alt + E` 只会提示先暂停童话事件。
:::

## 5. 出现问题时

| 现象 | 先检查 |
| --- | --- |
| 显示“需要校准” | 两个区域是否都已完成 |
| 显示“屏幕布局已变化” | 是否修改了分辨率、缩放、主屏幕或显示器排列 |
| 查蛋解析失败 | 名称是否完整；身高、体重和单位是否都在选区内 |
| 童话无法开始 | 是否已选择目标、完成双区域校准、处理待复核事件 |
| 快捷键无响应 | 按键绑定页是否注册成功，是否被其他软件占用 |

仍无法解决时，转到[屏幕识别故障排查](/screen-recognition/troubleshooting)。
