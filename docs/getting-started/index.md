# 五分钟快速开始

完成以下步骤后，软件即可显示玩家位置。路线导航是独立功能，需要定位成功后再开启。

![追踪总览界面位置示意](/screenshots/v0.9.3/tracking-dashboard.png)

> 图中用于说明总览区域和操作入口；路线、路径和日志内容会随玩家环境变化。

## 1. 完整解压

将下载的压缩包完整解压到纯英文、较短且可写的目录，例如：

```text
D:\SIFT-Map-Tracker-v0.9.3
```

不要直接在压缩软件预览窗口中运行，也不要只复制客户端 EXE。客户端、Python 服务、地图、特征缓存和运行库必须保持原有目录结构。

## 2. 启动客户端

运行包体根目录中的客户端程序。第一次启动时选择定位算法；不确定时使用 **SIFT**。

## 3. 框选小地图

1. 将游戏切换到地图界面正常可见的状态。
2. 点击“框选截取范围”。
3. 框住游戏右上角完整小地图，不要包含任务文字或旁边按钮。
4. 使用“预览截取区域”检查结果。

框选范围会保存到 `sift-map-tracker/runtime/settings_override.json`。更换显示器、分辨率、缩放比例或游戏窗口模式后，应重新框选。

## 4. 启动追踪

点击“启动追踪”。日志出现服务启动信息后，实时轨迹视图和 HUD 会开始更新。

::: tip
启动追踪只负责识图定位，不会自动开始路线导航。
:::

## 5. 开始导航

选择路线后点击“路线导航开始”。需要查看全局进度时打开“导航总览”，默认快捷键为 `Alt + F8`。
