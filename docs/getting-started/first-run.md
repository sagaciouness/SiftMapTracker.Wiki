# 首次启动

## 算法选择

首次启动会要求选择定位算法：

- **SIFT**：默认推荐，地标和纹理较丰富时稳定性最好。
- **模板匹配（TM）**：适合固定比例和较清晰的小地图，但对缩放、旋转和重复纹理更敏感。

选择结果可在设置页修改，重新启动追踪服务后生效。

## 首次框选

首次启动追踪前必须在当前电脑上框选小地图。项目不会沿用开发者电脑的截图坐标。

框选时：

- 保留小地图的完整圆形或方形内容；
- 不要框入地图外的任务列表和按钮；
- 不要把区域缩得过小；
- 在多显示器环境中，将鼠标放到游戏所在屏幕再打开框选器。

## 如何判断成功

满足以下条件即可认为首次配置有效：

1. 预览窗口能持续显示小地图。
2. 服务状态为运行中。
3. 日志没有持续出现 Capture、OpenCV 或特征文件错误。
4. 在有明显道路和建筑的区域移动时，坐标和 HUD 指针随之更新。
