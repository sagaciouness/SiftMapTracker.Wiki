# 设置说明

设置写入 `sift-map-tracker/runtime/settings_override.json`。删除或重置该文件会恢复用户参数，但不会改变程序内置的地图原点。

## 截图区域

`MINIMAP_REGION` 包含左、上、宽和高，使用 Windows 虚拟桌面的物理像素坐标。宽高必须为正数且不超过安全上限，整个区域必须落在虚拟桌面范围内。

## 定位参数

- 算法选择：SIFT、TM 或 ORB；
- 状态输出间隔：默认 50 ms；
- SIFT 匹配比例、最小匹配点和 RANSAC 阈值；
- CLAHE 局部对比度增强；
- 路线吸附距离、静止死区和大跳变限制。

## HUD 设置

可调整地图缩放、透明度、进度点显示距离、箭头增强、锁定与点击穿透，并可重置灵动岛位置。

## 恢复默认

参数异常时优先使用客户端“恢复默认”，然后重新框选小地图并重启追踪。不要从其他玩家电脑直接复制截图区域坐标。
