# 快捷键

| 功能 | 默认快捷键 | 范围 |
| --- | --- | --- |
| 灵动岛锁定/穿透切换 | `Ctrl + Alt + H` | 全局 |
| 打开导航总览 | `Alt + F8` | 全局 |
| 框选截取范围 | `Ctrl + Shift + R` | 全局 |
| 临时解锁拖动灵动岛 | 按住 `Alt` | 应用内 |
| 确认多路线切换建议 | `Ctrl + Shift + F8` | 全局 |
| 忽略当前路线建议 | `Ctrl + Shift + F9` | 全局 |
| 在当前位置重新导航 | `Ctrl + Shift + F10` | 全局 |

除内置的 `Alt` 临时解锁外，其余标记为可编辑的快捷键可在快捷键页修改。

## 注册失败

如果状态显示注册失败，通常是其他程序占用了同一组合键。关闭冲突程序或换用新的组合键后保存。恢复默认是事务式操作：保存失败时会尽量恢复之前的有效快捷键。
