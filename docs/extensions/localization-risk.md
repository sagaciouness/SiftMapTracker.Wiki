# 定位风险图

定位风险图是随客户端打包的静态 PNG，用于估计不同地图区域的理论定位难度。

![S3 静态定位风险图](/screenshots/v0.9.3/localization-risk-map.png)

## 颜色含义

- 绿色：特征和纹理较丰富，通常较稳定；
- 黄色：需要注意，遮挡或缩放变化可能影响结果；
- 橙色：较高风险；
- 红色：高风险，常见于大面积草地、雪地、海面或重复纹理。

## 操作

- 开关风险图层；
- 调整风险图透明度；
- 鼠标滚轮缩放；
- 左键或中键拖动；
- 使用“适应窗口”和“100%”恢复常用视图。

::: info
风险图是使用固定参数离线估算的结果，不代表实时遮挡、特效、昼夜和游戏更新后的所有情况。
:::
