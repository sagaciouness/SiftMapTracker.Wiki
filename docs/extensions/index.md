# 功能拓展

功能拓展页提供不属于核心实时定位链路的附加能力。关闭拓展后，不应增加追踪负担。

![功能拓展页](/screenshots/v0.9.3/extensions-overview.png)

## 野生炫彩检测

从小地图中检测特定野生物品图标，经过连续帧确认后播放提示音并显示通知。检测状态在每次启用时重新开始，手动采集不会继承上一次候选帧数。

## 地图标记

打开独立的标记与导出工作台，将当前路线、资源缓存和玩家标记组合为图层。详见[地图标记](/editors/survey-map)。

## 定位风险图

查看开发阶段离线生成的静态 SIFT/TM 可定位性估算，不会在玩家电脑上运行额外分析。详见[定位风险图](/extensions/localization-risk)。
