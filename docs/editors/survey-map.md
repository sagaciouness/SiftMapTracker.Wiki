# 地图标记

地图标记页用于把路线、资源缓存和玩家标记组合成可编辑图层，并导出 PNG。它不参与实时追踪和导航。

## 默认图层顺序

打开新项目时自动加入当前资源和当前路线，顺序从上到下为：

1. 默认标记图层；
2. 当前路线图层；
3. 当前资源图层。

列表顶部即渲染顶部，路线会显示在资源缓存上方。路线和资源属于只读系统图层，可以改名、隐藏、锁定、调整透明度和排序，但不能编辑内部节点或资源点。

## 标记工具

- 选择与拖动
- 平移、滚轮缩放和中键拖动
- 点位与文字
- 自由画笔与橡皮
- 矩形、圆形和箭头
- 撤销与重做

绘制前必须选择未锁定的标记图层。文字、名称、颜色、粗细和透明度修改都会记录到撤销历史。

## 保存和导出

- 项目保存到 `sift-map-tracker/survey_maps`；
- 路线保存来源引用和快照，原路线变化后旧项目仍能稳定打开；
- 资源图层使用资源编辑器生成的 cache PNG，不重新解析资源点 JSON；
- 全图导出会裁切到地图及可见内容范围，并在底部标注使用的软件与作者信息；
- 当前视口导出保持所见即所得，不添加品牌区。

退出、打开其他项目或新建项目时，如有未保存修改，软件会要求保存、放弃或取消。
