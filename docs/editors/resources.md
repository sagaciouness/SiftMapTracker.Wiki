# 资源点编辑

资源点编辑器用于在统一地图坐标系中记录物品位置，并生成供路线页、HUD 和地图标记页复用的缓存图。

## 基本流程

1. 选择资源分类和物品。
2. 新建或载入资源 JSON。
3. 点击“开始绘制”，在地图上添加点位。
4. 拖动已有点位进行微调。
5. 点击“结束绘制”自动保存并重建缓存图。

## 缓存图

缓存图是把资源点预先绘制到 8192 地图坐标上的 PNG。其他页面加载缓存图时无需逐点解析和创建大量 WPF 控件。

- 首次生成或资源 JSON 改动后需要重建；
- 有效性由源文件哈希、长度、修改时间、点数和地图尺寸共同判断；
- 缓存写入使用临时文件和原子替换，读取失败时保留旧的可用位图；
- 缓存不存在不影响资源 JSON，本体点位不会因此丢失。

## 加载较慢时

1. 等待状态栏显示缓存完成，不要连续快速切换多个大资源集。
2. 确认缓存目录可写且未被安全软件持续扫描。
3. 检查资源 JSON 是否包含异常数量的重复点。
4. 如果有效缓存仍反复重建，提交 JSON、manifest 和日志信息。
