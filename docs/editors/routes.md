# 路线编辑

路线编辑器用于建立导航节点和节点之间的连线。编辑前请切换到单路线模式，多路线模式下路线编辑会被锁定。

## 基本流程

1. 新建路线或选择已有路线 JSON。
2. 填写路线名称。
3. 使用节点工具在地图上放置节点。
4. 使用连线工具按行进顺序连接节点。
5. 根据路线设置闭环、传送点或其他节点属性。
6. 保存路线并回到追踪总览验证。

## 编辑规则

- 路线名称来自 JSON 内的 `name`，查看文件不应自动改成文件名。
- 节点坐标使用 8192 地图坐标系，与显示地图、资源点和测绘工具一致。
- 保存采用同目录临时文件和原子替换，避免服务恰好读取到半写入 JSON。
- 删除路线编辑器中的路线文件前应先备份；普通“删除图层”不会删除原始路线文件。

## 分享前检查

- 路线名称和说明不包含私人信息；
- 起点附近能够成功吸附；
- 每个节点均有合理连线；
- 闭环路线的尾节点能回到起点；
- 在实际游戏中完整运行至少一圈。

可在 [MapForge](https://www.mapforge.top/packs) 下载或分享路线。
