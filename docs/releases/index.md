# 版本记录

## v0.9.3

- 修复追踪服务启动被系统策略阻止时的客户端恢复流程。
- 提高小地图框选的最小尺寸，拒绝明显过小的区域。
- 延续 v0.9.x 的多显示器物理像素区域校验、资源缓存性能优化和 S3 赛季资源。

## v0.9.1

- 修复 125% / 150% 缩放和多显示器环境下合法区域被误判越界。
- 截图服务改用物理像素虚拟桌面边界，支持负坐标。
- 首次启动追踪要求在当前电脑重新框选小地图。
- 运行中重新框选后自动重启服务。
- 补充玩家包设置默认值和 S3 赛季资源目录。

## 版本策略

正文只描述最新稳定版。旧版本只保留变化、迁移方式和已知问题；不为每个版本复制整套教程。
