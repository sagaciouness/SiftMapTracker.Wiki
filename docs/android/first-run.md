# 手机端首次启动

## 开始前

1. 先打开《洛克王国：世界》，进入能看到完整小地图的游戏画面。
2. 保持游戏在前台，再用系统小窗或分屏打开“SIFT 地图追踪”。
3. 不要先在软件全屏界面授权录屏，再切回游戏。

Android 16 已验证的可靠流程是“游戏已显示 + 软件小窗/分屏 + 整个屏幕采集”。

## 四步启动流程

### 1. 授权悬浮窗

点击“1 授权悬浮窗”，在系统设置中允许本软件显示在其他应用上，然后返回软件。

### 2. 开始采集画面

点击“2 开始采集画面”。系统弹出录屏授权时：

- 选择“整个屏幕 / Entire screen”；
- 不要选择游戏或本软件作为单个应用；
- 授权后立即返回游戏画面。

Android 14 及以上会尽量直接请求整个默认屏幕，但仍应检查系统弹窗内容。

### 3. 框选小地图

回到游戏并等待采集到真实画面后，点击“3 回到游戏后框选小地图”。框选器会显示冻结截图：

- 只框住完整小地图；
- 不要包含任务文字、按钮或屏幕边缘；
- 确认截图确实是游戏，而不是软件主页或黑屏；
- 取消框选不会覆盖上一次保存的区域。

### 4. 开启追踪 HUD

选择定位算法和路线后，点击“4 开启追踪 HUD”。HUD 会显示在游戏上方；按住 HUD 标题区域可拖动位置。

## 通知栏控制

部分系统点击小窗会强制展开为全屏，此时可下拉通知栏，直接使用：

- 开始采集
- 框选小地图
- 开启 HUD / 停止 HUD
- 停止采集

这些按钮不会重新打开主界面，但“开始采集”仍会显示系统录屏授权页。

## 判断是否成功

- 主页状态显示悬浮窗已授权、采集帧已就绪、小地图区域已保存。
- HUD 不再显示“等待采集”，能看到地图或定位状态。
- 在道路、建筑等明显地标附近移动时，玩家位置持续变化。
- 通知标题在追踪时显示“SIFT 地图追踪运行中”。
