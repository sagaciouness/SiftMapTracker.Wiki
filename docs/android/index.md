# 手机端（精简版）

![SIFT 地图追踪 Android 图标](/android/app-icon.png){ width=160 }

手机端将地图定位、路线显示和悬浮 HUD 直接运行在手机上，不需要连接电脑端。当前文档对应 **v0.9.0**。

::: info 版本定位
手机端是面向随身导航的精简版本，只保留定位、路线和悬浮 HUD 等核心功能。路线编辑、资源点编辑、地图标记与图片导出请使用电脑端。
:::

## 平台要求

- Android 10 或更高版本。
- 目前仅支持主流 64 位 Android 手机。
- 需要系统悬浮窗、屏幕采集和前台服务权限。
- Android 13 及以上建议允许通知，以便使用采集控制按钮。
- 游戏必须允许系统屏幕采集；受保护画面可能显示黑屏。

## 当前能力

| 功能 | 手机端 v0.9.0 |
| --- | --- |
| 定位算法 | SIFT、模板匹配（TM） |
| 小地图来源 | Android 系统“整个屏幕”采集 |
| 路线 | 内置路线、JSON 导入、路线总览 |
| HUD | 悬浮地图、路线、玩家位置和状态 |
| 地图标记/PNG 导出 | 不支持 |
| 资源点编辑/野生物品识别 | 不支持 |
| 电脑端编辑工具 | 不支持 |

:::
