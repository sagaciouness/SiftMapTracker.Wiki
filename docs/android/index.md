# Android 客户端

![SIFT 地图追踪 Android 图标](/android/app-icon.png){ width=160 }

Android 版将地图定位、路线显示和悬浮 HUD 直接运行在手机上，不需要连接 Windows 客户端。当前文档对应 **v0.9.0**。

## 平台要求

- Android 10（API 29）或更高版本。
- 仅支持 `arm64-v8a` 设备。
- 需要系统悬浮窗、屏幕采集和前台服务权限。
- Android 13 及以上建议允许通知，以便使用采集控制按钮。
- 游戏必须允许系统屏幕采集；受保护画面可能显示黑屏。

## 当前能力

| 功能 | Android v0.9.0 |
| --- | --- |
| 定位算法 | SIFT、模板匹配（TM） |
| 小地图来源 | Android 系统“整个屏幕”采集 |
| 路线 | 内置路线、JSON 导入、路线总览 |
| HUD | 悬浮地图、路线、玩家位置和状态 |
| 地图标记/PNG 导出 | 不支持 |
| 资源点编辑/野生物品识别 | 不支持 |
| ORB | 已移除 |

Android 版与 Windows 版共享 8192×8192 地图坐标和路线 JSON 基础格式，但界面、权限和运行方式互不依赖。
