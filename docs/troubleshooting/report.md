# 反馈问题

可稳定复现的问题请提交到 [GitHub Issues](https://github.com/sagaciouness/SiftMapTracker.Client/issues)。普通交流可使用 [QQ 频道](https://pd.qq.com/g/pd29655551)。

```markdown
### 系统与显示环境
Windows 11；1920×1080；缩放 125%；单/多显示器

### 使用算法
SIFT / TM / ORB

### 问题现象
请说明看到的状态和错误文字。

### 复现步骤
1.
2.
3.

### 预期结果

### 日志
请粘贴从相关操作开始到报错后的完整日志。
```

## 隐私处理

提交前请删除 Windows 用户名、私人目录、路线隐私信息、访问令牌、网盘提取码和聊天账号。不要上传整个 `runtime` 目录，除非已逐项检查内容。
