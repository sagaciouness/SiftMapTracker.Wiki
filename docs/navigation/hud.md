# HUD 与灵动岛

HUD 用于在游戏画面上方显示路线、玩家位置、下一节点和资源缓存图；灵动岛用于紧凑显示当前路线、提示和状态。

## 交互

- `Ctrl + Alt + H`：切换灵动岛锁定或点击穿透。
- 按住 `Alt`：临时解锁并拖动灵动岛。
- 设置页可重置灵动岛位置和调整地图缩放、透明度等显示参数。

## 显示坐标

HUD 玩家位置使用原始追踪坐标。路线吸附位置仅用于路线展示，避免资源点和真实位置随路线发生偏移。

## 常见现象

- **指针不动但坐标变化**：恢复默认刷新间隔并重启追踪，参见[算法与参数](/tracking/algorithms)。
- **灵动岛超出边框**：先重置位置；仍存在时提交截图、主题、缩放比例和窗口尺寸。
- **无法拖动**：确认灵动岛未处于锁定或点击穿透状态，按住 `Alt` 再拖动。
