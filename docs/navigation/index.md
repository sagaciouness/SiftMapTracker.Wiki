# 路线导航

## 单路线模式

单路线模式只使用当前路线文件。选择路线后启动导航，服务按路线节点和连线计算进度、下一节点和距离。

## 多路线模式

多路线模式可加载多个候选路线。软件发现更适合的路线时给出切换建议，默认不会在需要确认的场景中静默替换路线。

- `Ctrl + Shift + F8`：确认当前路线切换建议。
- `Ctrl + Shift + F9`：忽略当前建议，五分钟内不再提醒。
- `Ctrl + Shift + F10`：在当前位置重新开始导航。

## 常用流程

1. 在路线配置区选择导航模式和路线文件。
2. 启动追踪并等待定位成功。
3. 点击“路线导航开始”。
4. 使用 HUD 或导航总览观察下一节点和路线进度。
5. 偏离路线且无法继续时，使用“在此位置重新导航”。

停止追踪服务时，导航开关和 HUD 状态也应恢复为停止，避免客户端与服务状态不一致。
