# 参与文档维护

欢迎修正错别字、补充复现步骤或改进教程。功能和故障说明必须基于当前稳定版实际行为，不要把猜测写成结论。

## 提交要求

1. 一个提交只处理一个主题。
2. 新页面必须加入 `docs/.vitepress/config.mts` 的侧栏。
3. 截图不得包含用户名、私人路径、访问令牌或未经同意的聊天内容。
4. 错误排查必须给出适用版本、现象、原因、处理步骤和仍不成功时需要收集的信息。
5. 不提交游戏解包内容、模型权重、发布密钥或第三方受版权保护的完整素材。

## 截图命名

使用 `模块-步骤-状态.png`，例如：

```text
tracking-01-dashboard.png
tracking-02-select-region.png
editor-route-01-overview.png
```
